Wisconsin Breast Cancer — Hyper-Parameter Tuning

This is a short practical in which the computer is doing most of the work (hopefully).

We use the Wisconsin Breast Cancer dataset, focus on the hyper-parameter step only, and we will try to build the best possible model (in terms of accuracy score).

The steps involved are:

  • Setup notebook

Then try each of the following hyper-parameter optimisation frameworks:

  • GridSearch
  • optuna
  • hyperopt
  • ray-tune (TuneGridSearchCV)
  • ray-tune (TuneSearchCV using optuna)
  • ray-tune (TuneSearchCV using hyperopt)
